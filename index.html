OBJETIVOS DE ESTE OVI

Tweets by UniversidadUNAD

Objetivo general

El OVI Diseño de sitios web se elabora con la finalidad de poner en práctica todas aquellas herramientas básicas con las cuales podemos crear un sitio web online.

Objetivo Especifico 1

Dar a entender a los usuarios de este sitio web información básica y detalladas sobre qué es y cómo funciona GITHUB, GIT HTML5 y CSS3, esto se presenta por medio de varios ejemplos y videos explicativos con los cuales esperamos que al terminar cada usuario este en la capacidad de hacer su propio sitio web online básico.

Objetivo Especifico 2

Entender y comprender lo básico sobre el manejo de herramientas como git, github, html y css3 con las cuales se pueda efectuar el desarrollo de sitios web de lo básico a lo avanzado.

Objetivo Especifico 3

Desarrollar las temáticas abordadas en el curso de desarrollo de sitios web de una manera sencilla y práctica.
