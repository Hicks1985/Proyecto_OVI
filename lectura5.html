GITHUB PAGES

Imagen tomada de Hosting your website using Github Pages

GitHub es una plataforma de desarrollo colaborativo de software la cual permite alojar proyectos utilizando el sistema de control de versiones Git. El código se almacena de forma pública, aunque también se puede hacer de forma privada, creando una cuenta de pago.

También tiene una característica muy útil llamada Github Pages la cual te permite publicar código del sitio en vivo en la web.
